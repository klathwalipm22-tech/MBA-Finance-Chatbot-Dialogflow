## 📌 Project Overview

This project demonstrates the end-to-end development of an AI-powered Finance Chatbot — from intent design in Dialogflow ES, API integration via Google Cloud, to live deployment on Kommunicate with human handoff capability.

The chatbot covers 13 finance topics commonly studied in MBA Applied Finance programs, making complex financial concepts accessible through natural conversation.

## 💬 Chatbot Features

| Intent | Description |
|---|---|
| 📊 NPV Calculator | Explains Net Present Value with examples |
| 📈 IRR Intent | Explains Internal Rate of Return |
| 🏦 Loan Advisor | Personalized loan guidance |
| 💰 Investment Advisor | Investment planning basics |
| 🧾 Tax Saving Advisor | Tax-saving instruments guide |
| 📋 Budget Planner | Personal budgeting assistance |
| 💳 EMI Calculator | EMI calculation guidance |
| 📉 Break Even Intent | Break-even analysis explanation |
| 🏅 Credit Score Advisor | Credit score improvement tips |
| ⚖️ Current Ratio Intent | Liquidity ratio explanation |
| 📐 Debt Equity Intent | Leverage ratio concepts |
| 💹 ROI Intent | Return on Investment explained |
| 🛡️ Insurance Advisor | Insurance basics and guidance |
