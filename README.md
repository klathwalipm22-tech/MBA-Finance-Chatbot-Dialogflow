# Finance Chatbot using Dialogflow and Kommunicate

## Project Overview
AI-powered Finance Chatbot built using Dialogflow ES and deployed through Kommunicate.

## Technologies Used
- Dialogflow ES
- Google Cloud Platform
- Kommunicate
- Conversational AI

## Features
- NPV Calculator
- IRR Explanation
- Loan Advisor
- Investment Advisor
- Tax Saving Advisor
- Budget Planner
